🔧 Teknologi
  • HTML5 CSS3 Vanilla JS
  • LocalStorage Database (History & Codes)
  • GitHub Raw JSON Integration (Code Validation)
  • AtlanticH2H Real API - QRIS Payment Gateway
  • Vercel Static Deployment Ready
🔗 AtlanticH2H API Endpoints
POST https://atlantich2h.com/deposit/create
POST https://atlantich2h.com/deposit/status
POST https://atlantich2h.com/deposit/cancel
POST https://atlantich2h.com/get_profile
Content-Type: application/x-www-form-urlencoded
📁 Struktur File
  • dashboard.html - Halaman utama
  • qriscode.html - Input code → AtlanticH2H
  • qrisotomatis.html - Input nominal → AtlanticH2H
  • qrismanual.html - Foto QRIS statis (NO API)
  • dana.html / ovo.html / gopay.html - Transfer manual
  • qriscreate.html - Display QR + status check
  • detailsucces.html - Halaman sukses + Telegram
  • storage/ - Folder aset (logo, music, qris-manual.jpg)
⚠️ Catatan Penting
  • API Key AtlanticH2H terintegrasi di client-side
  • Untuk production maksimal, gunakan Vercel Serverless Functions untuk menyembunyikan API Key
  • Pastikan endpoint AtlanticH2H mengizinkan CORS
  • Upload file aset ke folder storage/ sebelum deploy
